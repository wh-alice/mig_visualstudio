# IDebugProgramHost2::GetHostId
Gets the process identifier of the process hosting this program.  
  
## Syntax  
  
```cpp#  
HRESULT GetHostId(   
   AD_PROCESS_ID* pdwId  
);  
```  
  
```c#  
int GetHostId(   
   AD_PROCESS_ID[] pdwId  
);  
```  
  
#### Parameters  
 `pdwId`  
 [in, out] An [AD_PROCESS_ID](../extensibility-debugger-reference/ad_process_id.md) structure that is filled in with the process identifier information.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
