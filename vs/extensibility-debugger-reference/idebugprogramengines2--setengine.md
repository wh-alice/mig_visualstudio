# IDebugProgramEngines2::SetEngine
Tells the program or program node which debug engine (DE) to use to debug this program.  
  
## Syntax  
  
```cpp#  
HRESULT SetEngine(   
   REFGUID guidEngine  
);  
```  
  
```c#  
int SetEngine(   
   ref Guid guidEngine  
);  
```  
  
#### Parameters  
 `guidEngine`  
 [in] The GUID of the DE.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
