# IDebugField::GetKind
This method gets the kind of field.  
  
## Syntax  
  
```cpp#  
HRESULT GetKind(   
   FIELD_KIND* pdwKind  
);  
```  
  
```c#  
int GetKind(  
   out enum_FIELD_KIND pdwKind  
);  
```  
  
#### Parameters  
 `pdwKind`  
 [out] Returns the kind of field as a combination of [FIELD_KIND](../extensibility-debugger-reference/field_kind.md) constants.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
