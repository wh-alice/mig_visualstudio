# Concurrency Namespace (Concurrency Visualizer)
The `Concurrency` namespace provides classes and functions that give you access to the Concurrency Runtime, a concurrent programming framework for C++. For more information, see [Concurrency Runtime](../Topic/Concurrency%20Runtime.md).  
  
## Syntax  
  
```  
namespace Concurrency;  
```  
  
## Members  
  
### Namespaces  
  
|Name|Description|  
|----------|-----------------|  
|[diagnostic Namespace](../profiling/diagnostic-namespace.md)|The `diagnostics` namespace provides functionality for emitting Concurrency Visualizer markers.|  
  
## Requirements  
 **Header:** cvmarkersobj.h
