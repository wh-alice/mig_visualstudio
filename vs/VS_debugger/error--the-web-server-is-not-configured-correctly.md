# Error: The web server is not configured correctly
Possible causes of this error include:  
  
-   Trying to debug a .NET Web application that was copied to a different machine, manually renamed, or moved.  
  
-   Not having enough IIS connections. For more information about deploying a web site to IIS, see [Create a Web Site](http://www.iis.net/learn/get-started/getting-started-with-iis/create-a-web-site).  
  
-   If you are trying to debug an ASP.NET application, please see [Publishing to IIS](https://docs.asp.net/en/latest/publishing/iis.html) for instructions on deploying to a remote computer running IIS 8 or higher, or [Remote Debugging ASP.NET on a Remote IIS 7.5 Computer](../VS_debugger/remote-debugging-asp.net-on-a-remote-iis-7.5-computer.md) for instructions on deploying to a remote computer running IIS 7.5.
