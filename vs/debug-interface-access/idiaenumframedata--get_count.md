# IDiaEnumFrameData::get_Count
Retrieves the number of frame data elements.  
  
## Syntax  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### Parameters  
 pRetVal  
 [out] Returns the number of frame data elements.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
