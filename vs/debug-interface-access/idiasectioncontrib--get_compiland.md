# IDiaSectionContrib::get_compiland
Retrieves a reference to the compiland symbol that contributed this section.  
  
## Syntax  
  
```cpp#  
HRESULT get_compiland (   
   IDiaSymbol** pRetVal  
);  
```  
  
#### Parameters  
 `pRetVal`  
 [out] Returns an [IDiaSymbol](../debug-interface-access/idiasymbol.md) object representing the compiland that contributed this section.  
  
## Return Value  
 If successful, returns `S_OK`. Returns `S_FALSE` if this property is not supported. Otherwise, returns an error code.
