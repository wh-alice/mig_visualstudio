# IDiaSymbol::get_dataKind
Retrieves the variable classification of a data symbol.  
  
## Syntax  
  
```cpp#  
HRESULT get_dataKind (   
   DWORD* pRetVal  
);  
```  
  
#### Parameters  
 `pRetVal`  
 [out] Returns a value from the [DataKind Enumeration](../debug-interface-access/datakind.md) enumeration specifying the kind of data such as global, static, or constant, for example.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns `S_FALSE` or an error code.  
  
> [!NOTE]
>  A return value of `S_FALSE` means that the property is not available for the symbol.
