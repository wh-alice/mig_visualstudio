# /SafeMode (devenv.exe)
Starts [!INCLUDE[vsprvs](../dv_TeamTestALM/includes/vsprvs_md.md)] in safe mode, loading only the default environment and services.  
  
## Syntax  
  
```  
devenv /SafeMode   
```  
  
## Remarks  
 This switch prevents all third-party VSPackages from loading when [!INCLUDE[vsprvs](../dv_TeamTestALM/includes/vsprvs_md.md)] starts, thus ensuring stable execution.  
  
## Description  
 The following example starts [!INCLUDE[vsprvs](../dv_TeamTestALM/includes/vsprvs_md.md)] in safe mode.  
  
## Code  
  
```  
Devenv.exe /SafeMode  
```
