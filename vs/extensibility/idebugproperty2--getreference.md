# IDebugProperty2::GetReference
Returns a reference to the property's value.  
  
## Syntax  
  
```cpp#  
HRESULT GetReference(  
   IDebugReference2** ppReference  
);  
```  
  
```c#  
int GetReference(  
   out IDebugReference2 ppReference  
);  
```  
  
#### Parameters  
 `ppRererence`  
 [out] Returns an [IDebugReference2](../extensibility/idebugreference2.md) object representing a reference to the property's value.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code, typically `E_NOTIMPL` or `E_GETREFERENCE_NO_REFERENCE`.
