# IDebugProgramDestroyEvent2::GetExitCode
Gets the program's exit code.  
  
## Syntax  
  
```cpp#  
HRESULT GetExitCode(   
   DWORD* pdwExit  
);  
```  
  
```c#  
int GetExitCode(   
   out uint pdwExit  
);  
```  
  
#### Parameters  
 `pdwExit`  
 [out] Returns the program's exit code.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
