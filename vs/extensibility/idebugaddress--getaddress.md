# IDebugAddress::GetAddress
Returns a structure describing an object and its location within its scope or container.  
  
## Syntax  
  
```cpp  
HRESULT GetAddress (  
   DEBUG_ADDRESS * pAddress  
);  
```  
  
```c#  
int GetAddress(  
   DEBUG_ADDRESS[] pAddress  
);  
```  
  
#### Parameters  
 `pAddress`  
 [in, out] A [DEBUG_ADDRESS](../extensibility/debug_address.md) structure that is filled in by this method.  
  
## Return Value  
 If successful, returns S_OK; otherwise, returns an error code.  
  
## Remarks  
 The [DEBUG_ADDRESS](../extensibility/debug_address.md) structure is passed to this method, which then fills it in with the appropriate information. How this information is interpreted depends on the kind of information returned and the symbol handler itself. See [DEBUG_ADDRESS](../extensibility/debug_address.md) for more details.
