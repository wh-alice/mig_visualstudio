# &#39;As&#39;, comma or &#39;)&#39; expected
The type parameter that has been supplied is lacking an `As`, comma (`,`), or closing parenthesis ( `)` ).  
  
 **Error ID:** BC32100  
  
### To correct this error  
  
-   Supply the missing `As`, comma, or closing parenthesis.
