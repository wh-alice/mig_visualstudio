# MSBuild Error MSB2005
**Element \<{0}> cannot contain attributes.**  
  
 The element specified in the message cannot contain attributes.  
  
### To correct this error  
  
-   Delete the attributes from the specified element.  
  
-   Check whether the project file has been modified or corrupted. If it has been modified or corrupted, open the project in the version of [!INCLUDE[vsprvs](../dv_TeamTestALM/includes/vsprvs_md.md)] in which it was created, save it, and then attempt to convert it again.
