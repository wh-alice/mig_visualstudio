# &#39;Is&#39; expected
A `TypeOf` operator occurs without an `Is` clause.  
  
 **Error ID:** BC30224  
  
### To correct this error  
  
-   Add an `Is` clause following the `TypeOf` operator; for example, `TypeOf MyControl Is ComboBox`.
