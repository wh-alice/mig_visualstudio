# IDiaEnumSegments::get_Count
Retrieves the number of segments.  
  
## Syntax  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### Parameters  
 pRetVal  
 [out, retval] Returns the number of segments.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
