# IDiaEnumInjectedSources::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Resets an enumeration sequence to the beginning.  
  
## Syntax  
  
```  
HRESULT Reset();  
```  
  
## Return Value  
 Returns `S_OK`.
