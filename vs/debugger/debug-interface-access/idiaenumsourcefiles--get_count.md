# IDiaEnumSourceFiles::get_Count
Retrieves the number of source files.  
  
## Syntax  
  
```cpp#  
HRESULT get_Count (   
   LONG* pRetVal  
);  
```  
  
#### Parameters  
 pRetVal  
 [out] Returns the number of source files.  
  
## Return Value  
 If successful, returns `S_OK`; otherwise, returns an error code.
